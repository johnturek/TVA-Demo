Session 3

APIM + MCP + Secure Auth

Presenter: JT

Why APIM?

The enterprise security layer

  • Auth validation — JWT token verification
  • Rate limiting — protect backend resources
  • Backend abstraction — swap implementations
  • Logging & analytics — full audit trail
  • CORS — controlled cross-origin access
APIM sits between Copilot Studio and your backend — every request is authenticated, logged, and rate-limited. No direct backend exposure.

MCP — Model Context Protocol

Structured tools for agent interoperability

MCP lets your agent call structured tools instead of free-form HTTP.

Workshop MCP Tools

  • help — list all tools & usage
  • get_my_profile — OBO demo (Graph API)
  • analyze_policy — compliance questions
🚨 Transport Warning:
Copilot Studio dropped SSE transport support. MCP servers must use Streamable HTTP transport.

OBO Token Flow

User identity end-to-end

sequenceDiagram
  participant U as 👤 User
  participant CS as Copilot Studio
  participant APIM as Azure APIM
  participant BE as Backend API

  U->>CS: Ask question
  CS->>APIM: Forward request + user token
  Note over APIM: validate-azure-ad-token<br/>JWT audience & issuer check
  APIM->>APIM: OBO token exchange
  APIM->>BE: Request + OBO token + user context
  BE-->>APIM: Response
  APIM-->>CS: Response
  CS-->>U: Answer with citations

Why OBO matters for TVA

  • Audit trail — NERC CIP requires logging who accessed compliance data
  • Role-based access — nuclear vs. grid clearance levels
  • User context — personalized to user's division

Production auth config

  • Entra ID V2 + federated credentials (secret-less)
  • validate-azure-ad-token APIM policy
  • Require users to sign in
  • Token refresh for long sessions

Deploy to Production

One command. ~15 minutes.

cd ~/TVA-Demo
npx just workshop:ship

This automatically:

  1. Creates Entra ID app registration with correct API permissions
  2. Builds & pushes MCP server to Azure Container Registry
  3. Deploys MCP server as a Container App (HTTPS endpoint)
  4. Provisions APIM with JWT validation policy
  5. Outputs all values needed for Copilot Studio
